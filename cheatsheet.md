# Minha Cheatsheet DevOps

## Linux Básico
pwd - ME MOSTRA ONDE ESTOU. EXIBE EXATAMENTE ONDE ESTOU NA ESTRUTURA.
 
ls -  ME LISTA TUDO QUE TENHO NO MEU DIRETÓRIO - ARQUIVO VCHAGAS
 
cd - NAVEGAR ENTRE PASTAS - SAIR E ENTRAR EM OUTRA FAMOSO MOUSE
 
mkdir - CRIADOR DE DIRETÓRIOS - PASTAS

touch - CRIADOR DE ARQUIVOS - SEMPRE VAZIOS PARA ALIMENTAR

cp - COPIAR ARQUIVOS DE UMA ORIGEM PARA UM DESTINO

mv - MOVER E RENOMEAR O ARQUIVO DIRETÓRIO
 
rm - EXLUIR ARQUIVOS E DIRETÓRIOS PERMANENTE
 
cat - MOSTRAR NA TELA O QUE TEM NO ARQUIVO, E CONCATENAR

nano -  EDITOR DE TEXTO DE ARQUIVOS CRIADOS
